# Farmer Crop–Climate Mismatch Early Warning System

## Problem
Climate change is making traditionally grown crops unsuitable in many regions, causing yield losses and economic hardship for farmers.

## Solution
An AI-powered system that analyzes soil conditions and climate trends to identify crop-climate mismatches and recommend climate-resilient alternatives.

## Features
- Crop recommendation using Machine Learning
- Neural Network-based prediction
- Climate risk analysis
- Economic impact estimation
- Water savings estimation
- Interactive Streamlit dashboard

## Datasets
- Crop Recommendation Dataset
- District Wise Rainfall Dataset

## Tech Stack
- Python
- Pandas
- Scikit-Learn
- TensorFlow
- Streamlit
